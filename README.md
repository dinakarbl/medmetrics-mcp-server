# MedMetrics MCP Server

A custom MCP server for exposing synthetic healthcare quality metrics, policy resources, and analysis tools to AI agents.

## Project Goal

The goal is to build a simple, secure, and testable MCP server that can provide healthcare-quality data and tools to AI applications.

## Planned Features

- Synthetic hospital quality metrics
- MCP tools for metric lookup and analysis
- MCP resources for policy and metric definitions
- MCP prompts for quality investigation workflows
- Input validation with Pydantic
- Logging and error handling
- Tests
- Docker support

## Tech Stack

- Python
- MCP
- Pydantic
- FastAPI
- PostgreSQL
- Docker
- Groq API
